# GadgetShop 🛍️ - Modern Mini E-commerce Platform

A fast, clean, and fully-responsive mini E-commerce web application built using **Next.js 14+ (App Router)**, **Tailwind CSS**, and **Zustand** for global state management. This project features real-time search, dynamic category filtering, smooth animations, a seamless shopping cart experience, and an interactive dynamic checkout flow.
Live Link: https://mellow-frangollo-3e2d36.netlify.app

---

## 🚀 Live Preview & Showcase

* **Responsive Layout:** Beautiful modern design optimized for Desktop, Tablet, and Mobile devices.
* **Smooth Animations:** Implemented micro-interactions using **Framer Motion**.
* **State Management:** Lightweight and ultra-fast state handling with **Zustand**.

---

## ✨ Features Breakdown

### 1. 🌌 Hero & Trust Banner
* Highly premium dark-themed promotional banner with dynamic text gradients.
* Trust-building section featuring global service badges (Free Shipping, Secure Payment, etc.).

### 2. 🔍 Real-Time Search & Category Filter
* **Smart Search:** Live global filtering on the product feed without page refreshes.
* **Category Badges:** Instant filtering between electronics, accessories, and fitness gears with layout animations.

### 3. 🛒 Advanced Shopping Cart System
* Add, remove, and track product quantities dynamically.
* Real-time cart counter embedded in the navigation bar with a subtle pulse effect.
* Complete order breakdown summary with automatic price subtotal calculations.

### 4. 💫 Micro-Animations & Interactions
* Hover-to-lift effects on product cards.
* Interactive tap-scale responses on action buttons.
* Layout-shifting animations powered by Framer Motion.

### 5. 🎉 Dynamic Checkout Flow
* Interactive "Proceed to Checkout" action routing to a beautifully animated order confirmation success screen.

---

## 🛠️ Tech Stack & Architecture

* **Framework:** Next.js (React Server Components & Client Side Interaction Architecture)
* **Styling:** Tailwind CSS (Utility-first CSS Framework)
* **State Management:** Zustand (Stores item states globally)
* **Animations:** Framer Motion
* **Icons:** Lucide React
